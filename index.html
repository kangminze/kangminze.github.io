git 使用指南

发表于 2018-04-21

添加一个项目到git仓库

1.进入已存在项目

cd xxx

2.初始化该项目，会在该目录下生成.git目录，本地私有仓库

git init

3.添加远程依赖，git地址为一个已存在地址

git remote add origin https://gitee.com/kxz10323/boss.git

4.pull下最新代码

git pull origin master

5.提交本地代码并push到远端仓库

git add .
git commit -m "初始化"
git push origin master

至此提交成功
